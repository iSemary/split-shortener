# Split-Shortener

Split-Shortener is a Url shortener with ads area included.
<br>
Created By @iSemary

## Installation

##### Clone the repo
```bash
git clone https://github.com/iSemary/split-shortener.git
```
##### Open folder
```bash
cd split-shortener
```
##### Install packages
```bash
composer install 
```
##### Migrate databases
```bash
php artisan migrate
```
