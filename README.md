## Installation

##### Clone the repo
```bash
git clone https://github.com/iSemary/split-shortener.git
```
##### Open folder
```bash
cd split-shortener
```
##### Install packages
```bash
composer install 
```
##### Migrate databases
```bash
php artisan migrate
```
